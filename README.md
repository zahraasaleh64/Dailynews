# Daily News App

A modern Flutter application for reading and managing news articles.

## Features
- **Browse News**: View the latest news articles with high-quality images.
- **Detailed View**: Read the full content of any article.
- **Search**: Find specific articles quickly.
- **Admin Management**: Add new categories and publish news articles with image uploads.

## Technology Stack
- **Frontend**: Flutter (Mobile & Web)
- **Backend**: PHP (REST API)
- **Database**: MySQL

## Setup
1. Clone the repository.
2. Run `flutter pub get`.
3. Configure the `_baseURL` in your Flutter files to point to your PHP server.
4. Upload files in the `phpFiles` directory to your web server.
